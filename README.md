# tchebit-releases

Public release binaries for Tchê Bit client MCP servers.

## tchebit-news-mcp (client edition)

Semantic search over Tchê Bit news, newsletters, and premium newsletters —
embeds and caches locally, no server-side search. Download the binary for
your platform from the [Releases](../../releases) page.

Requires an MCP key from your Tchê Bit account (Settings → MCP Keys), set
as `NEWSLETTER_API_TOKEN`.
